<Section icon={Cpu} title="Requisitos de Sistema de Guildrun: Especificações de PC">

Antes de mergulhar na profundidade estratégica deste autobattler roguelike PvE single-player, verificar se o hardware do seu PC está alinhado com os **requisitos de sistema oficiais de Guildrun** é o primeiro passo para uma experiência de jogo fluida. Desenvolvido pela Leyline, o jogo apresenta cálculos de combate complexos, rastreamento de status e efeitos visuais vibrantes que exigem um hardware estável para evitar lentidões no fim de jogo. 

Como os autobattlers processam centenas de cálculos simultaneamente — como efeitos de relíquias ativas, escalonamento de atributos e especializações individuais de heróis —, ter um processador capaz é tão importante quanto a sua placa de vídeo. Abaixo estão as especificações de hardware mínimas e recomendadas para rodar o jogo de forma eficiente.

### Requisitos Mínimos do Sistema

As especificações mínimas foram projetadas para atingir a resolução de 1080p a 30 a 45 quadros por segundo durante confrontos de combate padrão. Esta configuração é ideal para setups de baixo custo, PCs de mesa mais antigos e notebooks de trabalho que utilizam gráficos integrados.

| Componente | Especificação Mínima |
| :--- | :--- |
| **SO** | Windows 10 de 64 bits |
| **Processador** | Intel Core i3-6100 / AMD FX-4350 |
| **Memória** | 4 GB de RAM |
| **Placa de Vídeo** | NVIDIA GeForce GTX 660 / AMD Radeon HD 7850 |
| **DirectX** | Versão 11 |
| **Armazenamento** | 2 GB de espaço disponível |

### Requisitos de Sistema Recomendados

Para desfrutar do jogo a cravados 60 quadros por segundo com efeitos de partículas de alta qualidade, tempos de carregamento rápidos e sem travamentos durante as intensas rodadas de Modo Infinito no fim do jogo, é altamente recomendável visar as especificações recomendadas.

| Componente | Especificação Recomendada |
| :--- | :--- |
| **SO** | Windows 10/11 de 64 bits |
| **Processador** | Intel Core i5-4460 / AMD Ryzen 3 1200 |
| **Memória** | 8 GB de RAM |
| **Placa de Vídeo** | NVIDIA GeForce GTX 960 / AMD Radeon RX 470 |
| **DirectX** | Versão 11 |
| **Armazenamento** | 4 GB de espaço disponível |

<Callout type="warning" title="Necessário SO de 64 Bits">
O jogo não pode ser iniciado em versões de 32 bits do Windows. Certifique-se de que seu sistema operacional esteja totalmente atualizado para uma arquitetura de 64 bits antes de tentar instalar o cliente de jogo.
</Callout>

</Section>

<Section icon={Wrench} title="Otimização Gráfica e Dicas de Desempenho">

Se o seu PC estiver mais próximo das especificações mínimas de hardware, você poderá notar pequenas quedas de quadros quando várias especializações de alto nível ativarem seus efeitos visuais simultaneamente. Felizmente, o menu de configurações do jogo oferece várias opções para recuperar o desempenho perdido sem sacrificar a clareza visual.

Ao ajustar sistematicamente as configurações que consomem muitos recursos, você pode manter uma taxa de quadros estável mesmo durante rodadas caóticas no fim do jogo, onde a tela fica cheia de invocações, indicadores de status e feitiços de área de efeito.

<Steps>
  <Step num="1" title="Configurar Modo de Exibição">
    Altere de Tela Cheia Exclusiva para Sem Bordas em Janela. Isso reduz a latência de entrada e evita travamentos se você costuma usar Alt-Tab com frequência para verificar listas de melhores heróis ou guias de builds.
  </Step>
  <Step num="2" title="Reduzir Resolução de Sombra">
    As sombras representam o maior consumo individual de GPU no jogo. Reduzir a qualidade das sombras de Alta para Média ou Baixa pode aumentar sua taxa média de quadros em até 15% em placas de vídeo mais antigas.
  </Step>
  <Step num="3" title="Limitar Taxa Máxima de Quadros">
    Sincronize o limite da sua taxa de quadros diretamente com a taxa de atualização do seu monitor. Se você estiver jogando em uma tela padrão de 60Hz, limitar o jogo a 60 FPS evita o estresse desnecessário da GPU e o estrangulamento térmico.
  </Step>
  <Step num="4" title="Desativar Tremor de Tela">
    Desative a opção de tremor de tela no menu de acessibilidade. Isso não apenas melhora o desempenho durante sequências rápidas de ataque, mas também torna muito mais fácil acompanhar as barras de vida das unidades durante o combate.
  </Step>
</Steps>

<Callout type="success" title="Desempenho de Gráficos Integrados">
Seguir estas etapas de otimização permite que chips gráficos integrados modernos, como AMD Radeon 680M ou Intel Iris Xe, mantenham facilmente cravados 60 FPS na resolução 1080p.
</Callout>

</Section>
